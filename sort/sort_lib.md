# sort library
코딩테스트에 sorting문제가 나왔을 경우,  
직접 sort 알고리즘을 짜지 않고 라이브러리나 패키지를 사용하는 것도 한 방법이다.  
python의 sorted함수는 tim sort 알고리즘을 사용한다.  
c++의 std::sort는 quick sort + heap sort를 사용한다.  
std::stable_sort는 merge sort와 유사한 방법을 사용한다.
